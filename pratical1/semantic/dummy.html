In CSS we have 3 ways to insert styles to html document.They are External,Internal and Inline CSS.In external css we can write styles outside of html document in another file and svaed it as CSS extensions.Each HTML page must include a reference to the external style sheet file inside the link element, inside the head section.In internal css we can write css property and values with in style element,inside head section.Lastly,Inline css are used to apply a unique style for a single element.
